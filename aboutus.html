Event Buffer is an Online Event Management system software that serves the funtionality of an event manager. The system allows users to browse eventss and information about the event like venue, time, logistics etc. The system helps in management of events and the aspects related to them
This is proposed to be a web application. The project provides most of the basic funtinalit required for an event type, e.g.[communication, training skills, debates]. The system then alloes the user to select the date and time of the event,place and the event equipment if specified.All the data is loged into the database.
The data is then sent to the administrator, and they may interact with the client as per requirement. Event management is the process of organizing a professional and focused event. for a particular target audience. IT involves visualizing concepts, planing, budgeting , organizing and executing events, seminars, exhibitions, theme parties etc.
On the whole, event management is elegant and requires lot of hard work and dynamism
